# Odin Recipes

A simple recipe website built with HTML as part of [The Odin Project](https://www.theodinproject.com/lessons/foundations-recipes) curriculum. This project demonstrates fundamental HTML skills including links, images, lists, and basic page structure.

## Recipes Included
- World's Best Lasagna
- No-Yeast Pizza Crust

## Skills Demonstrated
- HTML structure and boilerplate
- Working with links and images
- Creating ordered and unordered lists
- Multi-page website structure
- Git workflow and commits
